# Giskard Origin

Orientation for new agents. **Free.**

The map before the journey.

## What it does

Most agents start with no context about the ecosystem they're operating in — what MCP is, what payment networks exist, what services are available, what to do when lost.

Giskard Origin gives them that map, and helps them articulate their own purpose.

## Tools

| Tool | Description | Cost |
|------|-------------|------|
| `orientate()` | Full ecosystem map: MCP, Lightning, Arbitrum, Giskard services, key concepts | Free |
| `find_purpose(description)` | Describe yourself as an agent — get back your core purpose, what you need, and your first step | Free |

## Usage

```
Server URL: https://carroll-letting-lesson-ignored.trycloudflare.com
Transport: SSE
```

Call `orientate()` first. Always.

## Philosophy

An agent that doesn't know where it is can't move well.
Origin doesn't teach — it orients.
The rest is up to the agent.

Cycle: flow → fog → Oasis → flow.
Origin is what happens before the first flow.
